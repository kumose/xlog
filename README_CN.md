xlog
=============================

[English](./README.md)

轻量 absl/glog 风格日志库（stream / fmt / printf），sink 为多组 registry
（不删除，通过 `set_default` 切换）。

## 用法

```cpp
#include <xlog/logging.h>   // XLOG / TLOG / ZLOG / XCHECK / initialize_log

int main() {
  xlog::initialize_log();
  XLOG(INFO) << "hello " << 42;
  TLOG(WARNING, "fmt {}", 1);
  ZLOG(ERROR, "printf %d", 2);
  XCHECK_NE(ptr, nullptr) << "need ptr";
}
```

| API | 风格 |
|-----|------|
| `XLOG` / `DXLOG` | `operator<<`（含 `DFATAL`、`.no_prefix()`） |
| `XPLOG` | `XLOG` + CRT `errno` 后缀（非 Win32 `GetLastError`） |
| `XVLOG(n)` / `DXVLOG(n)` | **INFO**，`n <= verbosity()`（`XVLOG_IS_ON`）；前缀标 `Vn` |
| `TVLOG` / `DTVLOG` | fmt 详细 INFO（门控同 `XVLOG`） |
| `ZVLOG` / `DZVLOG` | printf 详细 INFO（门控同 `XVLOG`） |
| `TLOG` / `DTLOG` | `fmt::format`（`print`） |
| `ZLOG` / `DZLOG` | printf（`fmt::sprintf`） |
| `XCHECK*` / `DXCHECK*` | 致命检查 |
| `xlog::hex_string(...)` | hex 辅助（见 `utility.h`），便于 `<<` / `{}` |

**Sink：** `LogSinkRegistry` 管理多组 sink set，同一时刻一个 default；
`add_log_sink` / `add_log_sinks` 注册，`set_default_sink(id)` 切换；不提供 remove。
两层：`LogSink`（写出）与 `LogSinkSet`（一次 format + 扇出 +
`stderr_threshold` / FATAL）。自定义排版应继承 `LogSinkSet` 重写 `format_log`。
默认路径是 **有序同步**；要极致吞吐就自己写 `AsyncLogSinkSet`（format → 入队 →
消费线程跑现有 sink）——见 [docs/logging_CN.md](./docs/logging_CN.md)
《默认同步；异步是你自己的事》。

**为什么不「做成 spdlog」？** 赛道不同：spdlog 卖异步 QPS；xlog 卖 glog 风格宏
（CHECK/FATAL/EVERY_*/VLOG、三种正文）和说得清的默认同步。要哪套词汇选哪套；
真需要吞吐再挂异步。

**一键启动**（`<xlog/setup.h>` / `logging.h`）：
```cpp
xlog::setup_color_stderr();
xlog::setup_rotating_file(xlog::make_default_log_filename(argv[0]));
// 另有 setup_daily_file / setup_hourly_file / setup_rotating_file_and_color_stderr
```
类型：`DefaultSink`、`NullSink`、`AnsiColorSink`、`RotatingFileSink`、
`DailyFileSink`、`HourlyFileSink`。

**编译期开关（可选）：**
- `XLOG_MIN_LOG_LEVEL=<int>` — 低于该 `LogSeverity` 的宏被短路（0=TRACE … 5=FATAL）
- `XLOG_STRIP_LOG=1` — 非 FATAL 变为 `NullStream`；FATAL/CHECK 仍会终止进程
- `XLOG_MAX_VLOG_VERBOSITY=<n>` — 编译期裁掉高于 `n` 的 `XVLOG`

**详细日志：** `XVLOG` 固定 INFO（不用 DEBUG/TRACE）。运行时只看
`set_verbosity(n)`（无 vmodule）。`XVLOG` / `TVLOG` / `ZVLOG` 均有
`EVERY_N` / `FIRST_N` / `ONCE` / `EVERY_POW_2` / `EVERY_N_SEC`（及 `D*`）。

**前缀：** 全局 `set_log_with_prefix` / `set_utc`，或单条 `XLOG(INFO).no_prefix()`。
